# DOM – JavaScript

## Acceso y selección del DOM

1. Crea una página HTML con un título `<h1>` que tenga un identificador. Desde JavaScript, selecciona ese elemento y cambia su texto por "DOM cargado correctamente".

2. Crea tres párrafos con la misma clase. Desde JavaScript, selecciona todos los párrafos y cambia su color de texto a azul.

3. Selecciona el primer elemento `<p>` del documento con dos párrafos usando `querySelector` y modifica su contenido.

---

## Modificación de contenido y atributos

4. Crea un botón que, al pulsarlo, cambie el contenido de un `<div>` por un mensaje personalizado.

5. Crea una imagen y un botón. Al pulsar el botón, cambia el atributo `src` de la imagen por otra diferente.

6. Crea un campo de texto deshabilitado. Mediante un botón, elimina el atributo `disabled` para permitir la escritura.

---

## Estilos y clases CSS

7. Crea un párrafo y un botón. Al hacer clic en el botón, cambia el color de fondo y el tamaño de letra del párrafo.

8. Crea un `<div>` con una clase CSS inicial. Añade un botón que active y desactive una segunda clase usando `classList.toggle()`.

9. Comprueba si un elemento tiene una clase concreta y muestra el resultado por consola.

---

## Creación y eliminación de elementos

10. Crea un botón que, al pulsarlo, cree un nuevo párrafo con un texto definido y lo añada al final del documento.

11. Crea una lista `<ul>` vacía y un botón. Cada vez que se pulse el botón, se añadirá un nuevo elemento `<li>` numerado automáticamente.

12. Crea una lista con varios elementos y un botón que elimine el último elemento de la lista.

---

## Navegación por el DOM

13. Selecciona un elemento hijo y cambia el color de fondo de su elemento padre.

14. Dado un elemento seleccionado, modifica el texto de su hermano siguiente.

15. Recorre todos los elementos hijos de un `<div>` y cambia su color de texto.
